# YinYueTaiDownloader

YinYueTaiDownloader is a script to download .flv from [http://www.yinyuetai.com/](http://www.yinyuetai.com/) by ID without the `Yinyuetai Client`.

## Use

```python
python YinYueTaiDownloader.py
```
